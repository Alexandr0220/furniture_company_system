# Учебная практика — Неделя 1

Работа выполнена в рамках первой недели учебной практики.

В рамках задания был сделан простой backend-проект:
- создана база данных
- реализован сервер
- проект разбит на папки
- подготовлены отчёты и ER-диаграмма

Для разработки использовался Python и FastAPI.  
Этот вариант был выбран, так как он достаточно простой и удобный для проверки.

## Структура проекта
- app — основной код приложения
- docs — документация и ER-диаграмма
- report — отчёты по заданиям
- tests — тестовые файлы
- run.py — запуск сервера
- requirements.txt — зависимости
